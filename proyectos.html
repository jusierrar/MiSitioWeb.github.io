Sistema de Gestión de Cajas de Almacén

El Sistema de Gestión de Cajas de Almacén es una aplicación desarrollada en Python utilizando Tkinter, que permite a los usuarios administrar y controlar las cajas almacenadas en un almacén. Esta aplicación está diseñada para brindar una interfaz intuitiva y amigable, y se ha implementado con dos roles de usuario: administrador y colaborador.

Sistema de Gestión de Mesas e Inventario para Comercio

El Sistema de Gestión de Mesas e Inventario es una aplicación desarrollada en Excel con Visual Basic, diseñada para facilitar la gestión de un comercio. Este sistema permite llevar un control exhaustivo del inventario de productos, asignarlos a las mesas correspondientes y generar informes de ventas diarias. Además, ofrece la funcionalidad de agregar nuevos productos a medida que llegan o se introducen al comercio..
